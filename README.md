##Recipe Store

### Description:
    To store a number of paper based recipes inside
    a postgres database

### To Run:
- Start up a postgres database running in a Docker 
    container
- Run the following command
```java -jar target/RecipeStore-0.0.1.jar```

### Recipe Class:
- Name : String
- Category: String
- Size : String
- Ingredients: Array
- Method : String
